# PayNote / クレメモ

PayNote（クレメモ）は、クレジットカードなどのキャッシュレス決済で発生する「後日の口座引き落とし」を集計・管理するための iOS アプリです。

利用日、決済名、引き落とし口座、店名、分類タグ、支払回数を記録し、将来の口座引き落とし予定を把握しやすくします。

## 対象範囲

このアプリが対象とするのは、利用日とは別の日に口座から引き落とされる支払いです。

- クレジットカードの一括払い
- クレジットカードの二回払い
- 後日まとめて口座引き落としされるキャッシュレス決済

## 対象外

PayNote は、後日の引き落とし管理に特化したアプリです。次のような、即時に残高や現金が動く取引の管理は対象外です。

- 即日のデビット決済
- 現金払い
- 現金チャージ
- プリペイド残高への即時チャージ
- 家計簿のような日々の現金収支管理

## 主な機能

- 新しい決済やチャージの記録
- 最近の決済履歴の確認
- 口座引き落とし予定の集計
- キャッシュレス決済名の管理
- 引き落とし口座の管理
- よく使う店の管理
- 分類タグの管理
- JSON エクスポート
- 日本語・英語ローカライズ
- 旧 Core Data データから SwiftData への移行

## 支払予定の考え方

決済ごとに利用日と締め日・支払日をもとに、口座引き落とし予定日を計算します。

二回払いの場合は、支払い予定を 2 回分に分けて管理します。

## 開発環境

- Swift 6
- SwiftUI
- SwiftData
- iOS 18 以降
- iPhone / iPad 対応

## 補足

旧バージョンで提供していた一部機能は、現在の SwiftUI / SwiftData 版では対象外です。

- Dropbox バックアップ同期
- iCloud KVS
- Wi-Fi 経由データ転送
- パスワードロック
- CSV インポート / エクスポート
- 広告、アナライズ、内部課金
